# 🚦 Traffic Accident Analysis Using Unsupervised Learning

## 📖 Project Overview

This project analyzes traffic accident data using Unsupervised Machine Learning techniques to identify hidden patterns, accident hotspots, and risk groups. The goal is to uncover meaningful insights from accident records without predefined target labels.

By applying clustering techniques, the project groups similar accident cases and helps understand factors contributing to road accidents.

---

## 📊 Dataset

**Dataset:** `traffic_accidents.csv`

The dataset contains information related to road traffic accidents, including accident characteristics, location details, environmental conditions, and severity indicators.

### 📌 Features

* 🌦️ Weather Condition
* 🚦 Traffic Control Device
* 💡 Lighting Condition
* 🚗 First Crash Type
* 🛣️ Roadway Surface Condition
* ⚠️ Primary Contributory Cause
* 🚑 Injury Severity
* ⏰ Crash Hour
* 📅 Crash Day
* 📆 Crash Month

📥 Dataset Source:

https://www.kaggle.com/datasets/oktayrdeki/traffic-accidents
